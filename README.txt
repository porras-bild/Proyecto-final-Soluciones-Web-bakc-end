Sistema de pet shop -- PHP & MySQL

Este es un sistema creado para vender productos para las mascotas.

Funciones:
Registro y login de usuario
CRUD de productos
Dashboard
Roles y pestañas para usuarios

Lenguaje de programación: PHP
Base de Datos: MySQL
Framework:Bootstrap
librerias: Fpdf, PHPqrcode
lenguaje de marcado:CSS

Instalacion:
Abre xampp e inicia(start) APACHE y MySQL
Ve a admin en las dos opciones.
Importa la base d edatos a xampp PhpMyadmin
escribe o elige el nombre de la carpeta en la que este el sistema.

Requisitos previos:
xampp
PHP
